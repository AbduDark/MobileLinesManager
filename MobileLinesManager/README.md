# Mobile Lines Manager

نظام إدارة خطوط الموبايل للشركات المصرية الأربع (فودافون، اتصالات، وي، أورنج)

## المتطلبات

- **نظام Windows** (التطبيق يستخدم WPF ولا يعمل على Linux/Mac)
- **.NET 8.0 SDK** - [تحميل من هنا](https://dotnet.microsoft.com/download/dotnet/8.0)
- **Visual Studio 2022** أو **VS Code** مع C# extension

## كيفية تشغيل التطبيق

### الطريقة الأولى: من خلال Visual Studio

1. افتح ملف `MobileLinesManager.sln` في Visual Studio 2022
2. اضغط F5 أو اضغط على زر "Start" لتشغيل التطبيق
3. سيتم إنشاء قاعدة البيانات تلقائياً عند التشغيل الأول

### الطريقة الثانية: من خلال سطر الأوامر

```bash
cd MobileLinesManager
dotnet run
```

## معلومات الدخول الافتراضية

عند التشغيل الأول، سيتم إنشاء حساب مسؤول افتراضي:

- **اسم المستخدم**: `admin`
- **كلمة المرور**: `admin123`

⚠️ **تحذير**: يرجى تغيير كلمة المرور بعد أول تسجيل دخول!

## موقع قاعدة البيانات

يتم حفظ قاعدة البيانات في:
```
%APPDATA%\MobileLinesManager\mobile_lines.db
```

على سبيل المثال:
```
C:\Users\YourUsername\AppData\Roaming\MobileLinesManager\mobile_lines.db
```

## المشغلون المدعومون

التطبيق يدعم أربعة مشغلين مصريين مع ألوان مخصصة:

| المشغل | اللون |
|--------|-------|
| فودافون | أحمر (#E60000) |
| اتصالات | أخضر (#00B140) |
| أورنج | برتقالي (#FF7900) |
| وي | بنفسجي (#6C2C91) |

## حل المشاكل الشائعة

### خطأ: "no such table: Operators"

هذا الخطأ يحدث عندما لا توجد قاعدة بيانات. الحل:
- قم بحذف ملف قاعدة البيانات: `%APPDATA%\MobileLinesManager\mobile_lines.db`
- أعد تشغيل التطبيق - سيتم إنشاء القاعدة تلقائياً

### مشاكل الكاميرا (QR Scanner)

التطبيق يستخدم مكتبة AForge للوصول إلى الكاميرا. إذا واجهت مشاكل:
- تأكد من أن الكاميرا متصلة وتعمل
- امنح التطبيق صلاحيات الوصول للكاميرا في Windows
